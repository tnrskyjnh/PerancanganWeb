Program 100 tablet untuk MAHASISWA

Universitas Jember mempunyai program baru bagi para mahasiswa Universitas Jember yaitu Program 1000 tablet. Bekerjasama dengan Huawei, Polytron, dan IM3, program ini diserahakan langsung oleh pihak rektorat kepada UKM Koperasi Mahasiswa Universitas Jember. Program ini bertujuan mempermudah mahasiswa untuk mengakses semua informasi yang ada dalam kampus Universitas Jember. Target untuk program ini adalah mahasiswa baru (angkatan 2015).
